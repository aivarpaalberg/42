# tagasivõtt \(undo\)

## Tähendus

{% hint style="info" %}
Funktsioon, mis võimaldab kasutajal tühistada kõige viimas\(t\)ena täidetud käsu \(käskude\) toime.
{% endhint %}

Mõned käsud ei ole tagasivõetavad.
