# täht \(letter\)

## Tähendus

{% hint style="info" %}
kirjamärk \(_graphic character_\), mida üksikuna või koos teistega kasutatakse eeskätt suulise kõne mingi helielemendi esituseks.
{% endhint %}

## Täpsemalt

Diakriitilised märgid eraldi kasutatuna ja kirjavahemärgid ei ole tähed.
