# abs\(\)

## Üksikasjalikumalt

[Argumendiks](../../terminid/sonastik/argument.md) võib olla täisarv \(_int_\), ujukomaarv \(_floating point number_\). Kui argumendiks on kompleksnumber \(_complex number_\) siis tagastatakse selle magnituud \(_magnitude_\)

```python
>>> abs(-42)
42
>>> abs(-4.2)
4.2
```
